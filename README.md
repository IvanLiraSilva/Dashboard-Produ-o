# Dashboard Semanal de Produção

Este repositório contém o Dashboard Semanal de Produção, criado no Power BI, que permite acompanhar os principais indicadores de desempenho de produção de forma interativa. O painel inclui métricas como Horas Trabalhadas, Horas Paradas, Eficiência, OEE (Overall Equipment Effectiveness) e outros dados cruciais para monitorar o processo produtivo.

## Como usar o Dashboard

O dashboard é interativo e permite que você filtre os dados de acordo com:

Linha de Produção

Período (por dia)

## Indicadores no Dashboard

Horas Disponíveis: Total de horas disponíveis para produção.

Horas Trabalhadas: Total de horas efetivamente trabalhadas.

Horas Paradas: Total de horas de parada.

Eficiência: A eficiência da produção baseada no total de horas trabalhadas.

OEE (Overall Equipment Effectiveness): O desempenho geral do equipamento, calculado a partir de Disponibilidade, Performance e Qualidade.

Pareto de Tempo X Tipo de Parada: Um gráfico de Pareto que mostra a distribuição do tempo perdido por tipo de parada.

Programado x Realizado: Comparação entre a produção programada e a produção realizada por dia (em fardos).

## Filtragem Interativa

Linha de Produção: Você pode selecionar diferentes linhas de produção (ex: 10pl, 10pp, 5, etc.) para visualizar os dados específicos daquela linha.

Período (Dia): Selecione um período específico (ex: 22/09/2025, 23/09/2025, etc.) para analisar os dados de um dia específico.

## Como Navegar no Dashboard

Filtros: Utilize os filtros para escolher a linha de produção e o período desejado.

No painel do lado direito, você pode selecionar a linha e o período usando os menus suspensos.

Visualizações: Os gráficos interativos permitem analisar a eficiência, a comparação entre produzido e programado, o Pareto de paradas e o OEE.

Interatividade: Passe o mouse sobre os gráficos para ver mais detalhes e métricas específicas de cada indicador.

## Resultados e Insights

Com o uso deste painel, você pode obter insights como:

Identificar as principais causas de paradas de produção (ex: paradas programadas, paradas por setup, manutenção corretiva).

Comparar a produção programada com a produção realizada para verificar a eficiência e os desvios.

Acompanhar o OEE para avaliar a disponibilidade, performance e qualidade dos processos de produção.
